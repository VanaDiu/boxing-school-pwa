# Школа бокса — PWA-приложение

Это полный архив учебного мобильного PWA-приложения для проекта «Школа бокса».

## Что внутри

- `index.html` — главная страница приложения.
- `styles.css` — оформление и адаптация под телефон.
- `app.js` — меню, форма заявки, сохранение заявки в localStorage, регистрация Service Worker.
- `manifest.json` — настройки PWA-приложения: название, иконки, цвет темы, режим запуска.
- `sw.js` — Service Worker для кэширования и offline-режима.
- `offline.html` — страница при отсутствии интернета.
- `icons/` — иконки приложения разных размеров.
- `backend/` — пример PHP-обработчика заявки и подключения к MySQL.
- `database/database.sql` — пример структуры базы данных.
- `screenshots/` — папка для скриншотов проверки приложения.

## Как открыть

1. Распаковать архив.
2. Открыть папку в Visual Studio Code.
3. Установить расширение Live Server.
4. Открыть `index.html`.
5. Нажать `Open with Live Server`.

## Как проверить PWA

1. Открыть сайт через Live Server.
2. Нажать F12 → Application.
3. Проверить Manifest и Service Workers.
4. В браузере появится возможность установить приложение.
5. После установки приложение откроется как отдельное мобильное/настольное приложение.

## Как подключить серверную часть

Для обычного просмотра PHP не нужен.  
Если нужно сохранять заявки в MySQL:

1. Установить XAMPP/OpenServer.
2. Создать базу через `database/database.sql`.
3. Настроить `backend/db_config.php`.
4. Отправлять форму на `backend/signup.php`.
